# waf-scan

> 前言：基于wafw00f的插件内容，实现多线程批量识别版本。

## Usage

```yaml
# 对单个目标进行扫描 #
python waf-scan.py -t https://example.com

# 对多个目标进行扫描 #
python jws-cli.py -f targets.txt

# 结果输出 #
`/reports/`目录下会生成对应目标的json结果文件。
```

## 程序兼容性

python版本 >=  3.8.0
